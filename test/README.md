## Dependencies

- Node.js
- Express
- morgan
- EJS
- bcrypt
- cookie-session


## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command (or `npm start` command if using nodemon).
